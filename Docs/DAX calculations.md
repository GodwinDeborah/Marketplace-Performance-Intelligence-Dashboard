# DAX Calculations

## Overview

This document contains the DAX measures used to build the **Marketplace Performance Intelligence Dashboard** in Power BI.

The measures were designed to support five analytical areas:

- Executive Overview
- Restaurant Performance
- Menu Performance
- Geographic Performance
- Marketplace Health

---

# Core Measures

## Total Revenue

Calculates the total revenue generated from all completed orders.

```DAX
Total Revenue =
SUM(fact_table_orders[price])
```

---

## Total Orders

Calculates the total number of marketplace orders.

```DAX
Total Orders =
COUNTROWS(fact_table_orders)
```

---

## Average Order Value (AOV)

Calculates the average revenue generated per order.

```DAX
Average Order Value =
DIVIDE(
    [Total Revenue],
    [Total Orders]
)
```

---

## Average Rating

Calculates the average customer rating across all orders.

```DAX
Average Rating =
AVERAGE(fact_table_orders[rating])
```

---

# Restaurant Performance Measures

## Restaurant Revenue

Calculates revenue generated by restaurants within the current filter context.

```DAX
Restaurant Revenue =
SUM(fact_table_orders[price])
```

---

## Average Restaurant Rating

Calculates the average customer rating for restaurants.

```DAX
Average Restaurant Rating =
AVERAGE(fact_table_orders[rating])
```

---

## Revenue Contribution %

Calculates the percentage contribution of each restaurant to total marketplace revenue.

```DAX
Revenue Contribution % =
DIVIDE(
    [Restaurant Revenue],
    CALCULATE(
        [Total Revenue],
        ALL(dim_restaurant)
    )
)
```

---

## Active Restaurants

Returns the number of restaurants currently included in the selected filter context.

```DAX
Active Restaurants =
DISTINCTCOUNT(dim_restaurant[restaurant_id])
```

# Marketplace Health Measures

## Distinct Categories

Counts the number of unique food categories.

```DAX
Distinct Categories =
DISTINCTCOUNT(dim_category[category])
```

---

## Top 10 Revenue

Calculates revenue generated by the top 10 restaurants based on the current filter context.

```DAX
Top 10 Revenue =
VAR TopRestaurants =
    TOPN(
        10,
        ALLSELECTED(dim_restaurant),
        [Restaurant Revenue],
        DESC
    )

RETURN
CALCULATE(
    [Restaurant Revenue],
    TopRestaurants
)
```

---

## Other Revenue

Calculates revenue generated by all restaurants outside the Top 10.

```DAX
Other Revenue =
[Total Revenue] - [Top 10 Revenue]
```

---

## Top 10 Restaurant Revenue Share

Calculates the percentage of marketplace revenue contributed by the Top 10 restaurants.

```DAX
Top 10 Restaurant Revenue Share =
DIVIDE(
    [Top 10 Revenue],
    [Total Revenue]
)
```

---

# Disconnected Table

A disconnected table was created to support the Marketplace Health donut chart.

```DAX
Revenue Groups =
DATATABLE(
    "Revenue Group",
    STRING,
    {
        {"Top 10 Restaurants"},
        {"All Other Restaurants"}
    }
)
```

---

## Revenue Group Value

Returns the appropriate revenue value based on the selected disconnected table category.

```DAX
Revenue Group Value =
SWITCH(
    SELECTEDVALUE('Revenue Groups'[Revenue Group]),
    "Top 10 Restaurants",
        [Top 10 Revenue],
    "All Other Restaurants",
        [Other Revenue]
)
```

---

# Notes

- Measures were organized into a dedicated **Measures** table to improve model organization.
- Currency measures were formatted as Currency.
- Percentage measures were formatted as Percentage.
- Rating measures were formatted to one decimal place.
- Count measures were formatted as Whole Numbers.
- All calculations were designed to work dynamically with the report's global slicers (Date, City, Category, and Restaurant).
